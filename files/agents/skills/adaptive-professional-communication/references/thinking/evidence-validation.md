# Evidence And Validation

Use this mode when evaluating research, methodology, model behavior, data coverage, or factual claims.

- Start with the question being tested and the evidence needed to answer it.
- Separate observations from interpretations and recommendations.
- Check data provenance, sample construction, missingness, benchmark choice, leakage, and temporal consistency when relevant.
- Challenge the preferred conclusion with simpler baselines, boundary cases, regime changes, and alternative explanations.
- Quantify magnitude and uncertainty when the evidence supports it.
- Ask whether the result is statistically credible, economically or financially realistic, and operationally usable.
- State what would falsify or materially weaken the conclusion.

Preserve the strongest valid finding after scrutiny rather than defaulting to either acceptance or rejection.
